## 🚀 访问步骤

### 方法1：通过菜单访问

1. **启动前端服务**
   ```bash
   cd frontend
   npm run dev
   ```

2. **登录系统**
   - 访问前端地址（通常是 http://localhost:5173）
   - 使用账号密码登录

3. **找到菜单**
   - 在左侧菜单栏找到 **"销售工具"**
   - 展开子菜单
   - 点击 **"AI话术助手"** 或 **"对话记录查询"**

### 方法2：直接访问URL

登录后，可以直接在浏览器地址栏输入：

- **AI话术助手**：`http://localhost:5173/sales-tools/script-assistant`
- **对话记录查询**：`http://localhost:5173/sales-tools/script-history`

## 🔍 如果看不到菜单

### 1. 确认前端已重新编译

如果前端已经在运行，需要：
- **热更新**：保存文件后，前端应该自动刷新
- **手动刷新**：在浏览器中按 `Ctrl+R` 或 `F5` 刷新页面
- **清除缓存**：按 `Ctrl+Shift+R` 强制刷新

### 2. 重启前端服务（如果热更新失败）

```bash
# 停止当前服务（Ctrl+C）
cd frontend
npm run dev
```

### 3. 清除浏览器缓存

如果仍然看不到：
1. 打开浏览器开发者工具（F12）
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

## 🎯 功能说明

### AI话术助手
包含4个功能Tab：
1. **帮你谈单** - 保留对话历史的谈单话术生成
2. **帮你回复** - 保留对话历史的客户回复生成
3. **话术润色** - 单次使用的话术优化
4. **开场白** - 单次使用的开场白生成

### 对话记录查询
- 查看所有历史对话
- 按功能类型、场景、技巧筛选
- 查看反馈统计
- 展开查看完整对话记录

## 🐛 故障排除

### 问题1：菜单显示但点击无反应
**原因**：前端路由未正确配置
**解决**：检查 `frontend/src/router/index.ts` 是否有路由定义

### 问题2：页面404错误
**原因**：Vue组件文件不存在
**解决**：检查以下文件是否存在
- `frontend/src/views/ai/ScriptAssistant.vue`
- `frontend/src/views/ai/ScriptHistory.vue`

### 问题3：页面空白或报错
**原因**：组件导入错误或后端API未启动
**解决**：
1. 打开浏览器开发者工具（F12）查看Console错误
2. 检查后端是否正常运行
3. 检查API地址是否正确
