# 静态资源说明

## 图片资源

### Logo和头像
- `logo.png` - 应用Logo（建议尺寸: 200x200px）
- `default-avatar.png` - 默认用户头像（建议尺寸: 200x200px）

### TabBar图标
所有TabBar图标建议尺寸: 81x81px

- `tabbar/home.png` - 首页图标（未选中）
- `tabbar/home-active.png` - 首页图标（选中）
- `tabbar/customer.png` - 客户图标（未选中）
- `tabbar/customer-active.png` - 客户图标（选中）
- `tabbar/order.png` - 订单图标（未选中）
- `tabbar/order-active.png` - 订单图标（选中）
- `tabbar/stats.png` - 统计图标（未选中）
- `tabbar/stats-active.png` - 统计图标（选中）
- `tabbar/my.png` - 我的图标（未选中）
- `tabbar/my-active.png` - 我的图标（选中）

## 注意事项

1. 当前文件为占位文件，需要替换为真实的图片资源
2. 图标建议使用PNG格式，支持透明背景
3. 未选中状态建议使用灰色，选中状态使用品牌色（#3b82f6）
4. 可以使用在线图标生成工具或UI设计软件制作
