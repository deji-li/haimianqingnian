# 海棉青年 CRM 系统功能说明书

## 📋 项目概述

海棉青年 CRM 是专为学生教育培训机构设计的智能化客户关系管理系统，集成了 AI 分析、订单同步、营销自动化等先进功能，帮助教育机构实现客户获取、转化、服务的全流程数字化管理。

### 核心价值
- **AI 驱动**：深度学习算法赋能客户分析与营销内容生成
- **全流程覆盖**：从潜在客户到签约服务的完整业务闭环
- **数据智能**：实时数据同步与多维度分析报表
- **移动优先**：支持多终端应用，随时随地开展业务

---

## 🏗️ 系统架构

### 技术栈
- **后端**：NestJS + TypeScript + MySQL + Redis
- **前端**：Vue 3 + Element Plus + TypeScript + Vite
- **移动端**：UniApp 跨平台开发
- **AI 集成**：DeepSeek API + 百度 OCR

### 部署架构

## 💼 核心功能模块

### 1️⃣ 客户管理系统

#### 🎯 客户获取与分配
- **多渠道获客**：微信、电话、地推、转介绍等多种来源
- **智能分配**：基于销售能力、工作负荷自动分配客户
- **客户去重**：智能识别重复客户，避免资源浪费
- **批量导入**：支持 Excel 批量导入客户数据

#### 👤 客户画像分析
- **基础信息**：学生年级、家长角色、联系方式、地区分布
- **AI 智能分析**（20+ 维度）：
  - **经济水平评估**：消费能力、付费意愿分析
  - **需求痛点识别**：学习困难、关注重点、焦虑点
  - **兴趣特征分析**：学科偏好、活动参与、性格特点
  - **成交价值评估**：订单金额预测、成交周期判断
  - **跟进策略建议**：最佳沟通时间、话术推荐、优惠策略

#### 📈 客户生命周期管理
- **意向等级**：A 类（高意向）到 D 类（低意向）动态分级
- **阶段管理**：初步接洽 → 需求了解 → 体验参与 → 价格谈判 → 签约付款
- **智能提醒**：跟进时间提醒、生日提醒、重要事项提醒
- **流失预警**：基于行为分析预测客户流失风险

#### 📝 客户跟进记录
- **多维度记录**：电话沟通、微信聊天、上门拜访、活动参与
- **AI 聊天分析**：自动分析聊天记录，提取关键信息和销售机会
- **跟进计划**：制定个性化跟进计划，提高转化效率
- **语音转文字**：支持通话录音转文字，便于复盘分析

### 2️⃣ 订单管理系统

#### 📋 订单全生命周期管理
- **订单创建**：支持手动录入、小程序下单、系统同步多种方式
- **订单状态跟踪**：待付款、已付款、已排课、进行中、已完成、已取消
- **订单修改**：支持订单信息变更、课程调整、延期处理
- **订单查询**：多维度筛选查询，快速定位目标订单

#### 💰 佣金计算系统
- **销售佣金**：基于业绩阶梯、产品类型、客户类型计算
- **教师课时费**：按课程类型、时长、教师等级计算
- **团队提成**：团队主管可享受团队成员业绩提成
- **退费处理**：自动计算退费时的佣金调整

#### 🔄 外部系统同步
- **海棉系统对接**：实时同步订单数据，避免信息孤岛
- **同步状态跟踪**：成功、失败、重试机制，确保数据一致性
- **数据映射配置**：灵活配置字段映射关系
- **同步日志记录**：详细的操作日志，便于问题排查

### 3️⃣ AI 营销助手

#### ✨ 智能内容生成
- **6 大营销场景**：
  - **新客开拓**：首次接触话术、机构介绍文案
  - **跟进沟通**：针对性跟进话术、疑问解答
  - **活动推广**：体验课、讲座、促销活动文案
  - **转介绍引导**：老带新激励政策、推荐话术
  - **续费提醒**：课程到期提醒、续费优惠方案
  - **节日营销**：节日祝福、特色活动推广

- **个性化内容**：
  - 基于客户画像定制化生成
  - 融入客户痛点和需求特征
  - 结合知识库教育理念
  - 支持 12+ 种风格选择（专业、亲切、幽默等）

#### 📊 营销效果分析
- **内容质量评分**：AI 评估内容质量和预期效果
- **使用情况统计**：内容使用频率、客户反馈统计
- **A/B 测试**：不同内容版本效果对比
- **优化建议**：基于数据反馈的内容优化建议

#### 🎯 精准营销策略
- **客户分群**：基于 AI 分析结果进行精准分群
- **个性化推荐**：为不同客户群体推荐合适的内容
- **最佳时机预测**：预测最佳营销时间和渠道
- **转化漏斗分析**：营销活动全流程转化分析

### 4️⃣ 智能工具集

#### 🔍 百度 OCR 文字识别
- **证件识别**：身份证、学生证、户口本自动识别
- **文档处理**：合同、报名表、试卷等文档数字化
- **名片识别**：快速录入客户信息，建立人脉网络
- **批量处理**：支持批量图片识别，提高处理效率

#### 💬 智能聊天分析
- **会话质量评估**：分析销售沟通质量和专业度
- **客户意图识别**：自动识别客户需求和购买意向
- **关键信息提取**：提取客户信息、需求要点、承诺事项
- **销售技巧建议**：基于分析结果提供沟通技巧建议

#### ⚠️ 风险预警系统
- **客户流失预警**：基于行为模式预测流失风险
- **销售异常监控**：监控销售行为规范，避免违规操作
- **业绩风险提醒**：业绩目标达成风险预警
- **合规风险提示**：识别潜在合规问题，降低经营风险

### 5️⃣ 企业知识库

#### 📚 知识管理中心
- **知识分类管理**：产品知识、教学方法、常见问题、成功案例
- **智能标签系统**：AI 自动为知识内容打标签，便于检索
- **版本控制**：知识内容版本管理，追溯历史变更
- **权限管理**：基于角色的知识访问权限控制

#### 🔍 智能检索系统
- **全文搜索**：支持关键词、语义搜索
- **智能推荐**：基于用户行为推荐相关知识内容
- **知识图谱**：构建知识关联网络，发现潜在联系
- **快速定位**：在客户沟通中快速定位相关知识

#### 🧠 AI 知识挖掘
- **知识提取**：从文档、对话中自动提取有价值知识
- **知识整合**：将碎片化知识整合成系统化内容
- **质量评估**：AI 评估知识内容的准确性和实用性
- **持续学习**：基于用户反馈不断优化知识质量

### 6️⃣ 数据分析中心

#### 📈 实时数据大屏
- **关键指标监控**：新增客户、签约订单、业绩完成率
- **趋势分析图表**：业绩趋势、转化率变化、客户增长
- **团队排行**：销售业绩排行、团队贡献对比
- **预警提醒**：异常数据实时提醒，及时发现问题

#### 📊 多维度分析报表
- **销售漏斗分析**：从获客到签约的转化漏斗
- **客户价值分析**：客户生命周期价值、复购率分析
- **课程受欢迎度**：课程报名统计、满意度分析
- **教师绩效分析**：教师授课量、学员评价、业绩贡献

#### 🎯 目标管理系统
- **目标设定**：支持年度、季度、月度目标设定
- **进度跟踪**：实时跟踪目标完成进度
- **偏差分析**：分析目标偏差原因，制定改进措施
- **激励机制**：基于目标达成率的激励方案

## 🚀 系统特色功能

### AI 客户分析引擎
采用 DeepSeek 大语言模型，实现 20+ 维度客户智能分析：

#### 分析维度
1. **家庭背景分析**
   - 家长角色（父亲/母亲/其他监护人）
   - 教育程度和职业背景
   - 经济水平和消费观念
   - 教育投入意愿评估

2. **学生特征分析**
   - 年级和学习阶段
   - 学习能力和基础水平
   - 性格特点和兴趣爱好
   - 学习习惯和态度

3. **需求痛点识别**
   - 学习困难类型
   - 家长焦虑点
   - 关注重点排序
   - 价格敏感度分析

4. **成交价值评估**
   - 订单金额预测
   - 成交周期判断
   - 续费可能性评估
   - 转介绍潜力分析

5. **跟进策略建议**
   - 最佳沟通时间
   - 推荐沟通方式
   - 话术重点建议
   - 优惠策略推荐

### 智能营销内容生成
基于客户画像和知识库，生成个性化营销内容：

#### 内容类型
- **微信话术**：适合微信沟通的简短话术
- **电话脚本**：电话沟通的完整脚本
- **短信文案**：短信营销的简洁文案
- **邮件内容**：正式的邮件推广内容
- **朋友圈文案**：适合社交媒体分享的内容

#### 风格选择
- **专业型**：突出专业性和权威性
- **亲切型**：温暖贴心，拉近距离
- **幽默型**：轻松幽默，增加亲和力
- **励志型**：激励向上，传递正能量
- **数据型**：用数据说话，增强说服力
- **故事型**：通过案例故事引发共鸣

### 订单自动同步系统
与海棉系统深度集成，实现数据自动同步：

#### 同步功能
- **实时同步**：订单状态实时更新
- **双向同步**：支持数据的双向同步
- **异常处理**：同步失败自动重试
- **日志记录**：详细的同步操作日志

#### 数据映射
- **灵活配置**：可配置字段映射关系
- **数据转换**：支持数据格式转换
- **验证机制**：数据有效性验证
- **冲突处理**：数据冲突解决策略
